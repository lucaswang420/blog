## Project Overview

**CodeVerse | 技术宇宙** is a technical knowledge base blog built with Hexo static site generator and Butterfly theme. It hosts 100+ technical articles covering C++ development, system programming, frameworks, databases, web architecture, and DevOps.

## Common Commands

### Development
```bash
# Install dependencies
npm install

# Start local development server (http://localhost:4000)
npm run server
# or
hexo server

# Create a new post
npm run new "Post Title"
# or
hexo new "Post Title"
```

### Building & Deployment
```bash
# Clean generated files
npm run clean

# Generate static site
npm run build
# or
hexo generate

# Deploy to GitHub Pages
npm run deploy
# or
hexo deploy
```

### Auto-Post System
```bash
# Run auto-post system (publishes documents from unpost/ directory)
cd auto-post
python auto-post.py

# Quick run
python run.py
```

## Architecture

### Directory Structure
```
blog-source/
├── _config.yml              # Hexo main configuration
├── _config.butterfly.yml    # Butterfly theme configuration
├── package.json             # Node.js dependencies and scripts
├── source/                  # Source content
│   ├── _posts/             # Blog posts (organized by category)
│   ├── about/              # About page
│   ├── categories/         # Category pages
│   ├── tags/              # Tag pages
│   └── images/            # Static images
├── themes/butterfly/       # Butterfly theme
│   └── scripts/           # Theme customization scripts
│       ├── filters/       # Hexo filters (random_cover, post_lazyload)
│       ├── helpers/       # Theme helpers
│       └── common/        # Shared utilities
├── auto-post/             # Automated publishing system
│   ├── auto-post.py       # Main control script
│   ├── scripts/          # Processing scripts (reader, optimizer, reviewer, publisher)
│   └── config/           # System configuration
├── public/               # Generated static site (after hexo generate)
└── db.json              # Search database (auto-generated)
```

### Content Organization
Posts are organized hierarchically in [`source/_posts/`](source/_posts):
- `01-CPP核心语言/` - C++ core language features

- `03-开发工具/` - Development tools (build systems, debugging, version control)
- `05-框架库/` - Frameworks and libraries (Qt, gRPC)
- `06-数据库系统/` - Database systems (MySQL, PostgreSQL, Redis)
- `07-Web架构安全/` - Web architecture and security
- `08-云原生运维/` - Cloud native and DevOps
- `09-软件架构/` - Software architecture
- `99-参考手册速查/` - Reference materials

### Hexo Plugins
Key plugins configured in [`_config.yml`](_config.yml):
- `hexo-abbrlink` - Permanent link generation (algorithm: crc32, format: hex)
- `hexo-generator-searchdb` - Local search functionality
- `hexo-generator-json-content` - JSON content for search
- `hexo-deployer-git` - Git deployment to GitHub Pages
- `hexo-generator-feed` - RSS/Atom feed generation

### Butterfly Theme Customization
Theme configuration in [`_config.butterfly.yml`](_config.butterfly.yml):
- Visual effects: particles, starry sky, waves, neon glow
- Dark mode support with button toggle
- Custom code themes (github-light/github-dark)
- Canvas nest animation background
- Typewriter effect for subtitle
- Local search integration
- Giscus comments system

Theme scripts in [`themes/butterfly/scripts/`](themes/butterfly/scripts/):
- `filters/random_cover.js` - Random cover image generator for posts
- `common/default_config.js` - Default theme configuration

### Auto-Post System
Located in [`auto-post/`](auto-post/):
- **Purpose**: Automatically process and publish documents from `unpost/` directory
- **Processing Pipeline**:
  1. Reader: Document parsing
  2. Optimizer: Title and structure optimization
  3. Blog Enhancer: Add summaries, metadata
  4. Reviewer: Content review and corrections
  5. Privacy Checker: Detect sensitive content
  6. Publisher: Publish to Hexo blog

## Post Front Matter Format

All posts must include YAML front matter:
```yaml
---
tags:
  - Tag1
  - Tag2
title: Post Title
categories:
  - MainCategory
  - SubCategory
description: Brief description of the article content
abbrlink: 7a1e3650  # Auto-generated by hexo-abbrlink
date: 2025-12-04 22:47:58
---
```

Important fields:
- `abbrlink`: Auto-generated unique identifier (don't manually edit)
- `categories`: Use hierarchical format for proper organization
- `description`: Used for SEO and post previews

## Configuration Files

### [`_config.yml`](_config.yml) - Hexo Configuration
Key settings:
- `permalink: posts/:abbrlink/` - Uses abbrlink for URLs
- `theme: butterfly` - Active theme
- `language: zh-CN` - Chinese language
- `timezone: Asia/Shanghai` - Timezone
- Deployment to `git@github.com:lucaswang420/lucaswang420.github.io.git`

### [`_config.butterfly.yml`](_config.butterfly.yml) - Theme Configuration
Key features:
- Menu navigation (首页, 归档, 标签, 分类, 关于)
- Social links (GitHub)
- Code highlighting (github-light/github-dark themes)
- Background images and visual effects
- Local search enabled
- Giscus comments integration
